# 組み合わせ(パスカルの三角形)

## パスカルの三角形を用いて事前に組み合わせを計算する方法
以下のように事前に計算しておくと便利。($O(N^2)$)
```cpp
ll comb[61][61];

int main() {
    ll A, B, K;
    cin >> A >> B >> K;

    comb[0][0] = 1;
    for (int i = 0; i < 60; i++) {
        for (int j = 0; j <= i; j++) {
            comb[i + 1][j] += comb[i][j];
            comb[i + 1][j + 1] += comb[i][j];
        }
    }

    // パスカルの三角形を出力してみる
    for (int i = 0; i < 10; i++) {
        for (int j = 0; j <= i; j++) {
            cout << comb[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

```
