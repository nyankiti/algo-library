# bit全探索
`<<` という演算子はビット演算で、ビットを左へシフトする。よって、`(1<<n)`は10進数の$2^n$。
```cpp
// {0, 1, ..., n-1} の部分集合の全探索
for (int bit = 0; bit < (1<<n); ++bit) {
    vector<int> S;
    for (int i = 0; i < n; ++i) {
        if (bit & (1<<i)) { // 列挙に i が含まれるか
            S.push_back(i);
        }
    }
// 出力部分
    cout << bit << ": {";
    for (int i = 0; i < (int)S.size(); ++i) {
        cout << S[i] << " ";
    }
    cout << "}" << endl;
}
```
`n=3`の場合、標準出力は以下のようになる。
```
0: {}
1: {0 }
2: {1 }
3: {0 1 }
4: {2 }
5: {0 2 }
6: {1 2 }
7: {0 1 2 }
```
